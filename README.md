# 🎧 Lotify — Spotify Clone

Lotify is a modern **Spotify-inspired music streaming web application** built using React, Vite, and Tailwind CSS.
It delivers a clean, responsive, and fast user experience with core music player features and scalable frontend architecture.

---

## 🚀 Features

* 🎵 Music Player (Play / Pause / Seek)
* 📂 Playlist UI & Management
* ⚡ Fast Performance with Vite
* 🎨 Modern UI using Tailwind CSS
* 📱 Fully Responsive Design
* 🧩 Reusable Component Architecture

---

## 🛠️ Tech Stack

* **Frontend:** React + TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Package Manager:** npm

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/MANIKANDAN-17K/lotify.git
cd lotify/frontend
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run development server

```bash
npm run dev
```

👉 Open: `http://localhost:5173` (or shown port)
